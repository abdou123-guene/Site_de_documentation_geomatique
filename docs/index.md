#  Sources de données & métadonnées en géomatique

## 1. Pourquoi ces deux sujets sont indissociables ?

En géomatique, une donnée n’a de valeur que si l’on connaît son contexte :
- d’où elle vient,
- ce qu’elle représente,
- comment elle a été produite,
- comment elle peut être réutilisée.

Les **sources de données** définissent *le quoi* :
contenu, étendue, précision, fréquence de mise à jour.

Les **métadonnées** définissent *le comment* et *le pourquoi* :
provenance, qualité, méthodes de production, droits d’usage.

Bien gérés, ces deux volets rendent les projets :
- traçables,
- comparables,
- réutilisables,

tout en réduisant les coûts cachés (temps perdu, doublons, erreurs d’analyse).
