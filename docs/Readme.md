# <span style="color:blue;">1) Pourquoi les sources de données et les métadonnées sont indissociables</span>

En géomatique, une donnée n’existe vraiment que si l’on connaît :

- **d’où elle vient** (source, producteur, datation)
- **ce qu’elle représente** (définition, précision)
- **comment elle a été produite** (méthodes, traitements)
- **comment on peut la réutiliser** (licence, droits, formats)

-> Les **sources de données** définissent *le QUOI*.  
->Les **métadonnées** définissent *le COMMENT et le POURQUOI*.

Bien gérés, ces deux volets assurent :
- traçabilité,
- reproductibilité,
- comparabilité,
- réduction des erreurs et doublons,
- meilleure qualité analytique.

# <span style="color:blue;">2) Sources de données utilisées à l’AUDC</span>

## <span style="color:red;">1. INSEE</span> — Institut National de la Statistique et des Études Économiques
- Population : effectifs, structure, vieillissement/jeunesse.  
- Revenus : revenu médian, FiLoSoFi.  
- Flux : mobilités résidentielles & domicile–travail.  
- Indicateurs socio-économiques : emploi, chômage, catégories sociales.

---

## <span style="color:red;">2. SITADEL</span> — Données logement & construction
- Permis de construire.  
- Achèvements.  
- Logements créés/démolis.  
- Surfaces & typologies.  
- Dynamiques à l’échelle commune → EPCI → département.

---

## <span style="color:red;">3. BPE</span> — Base Permanente des Équipements
- Équipements : santé, sport, éducation, culture.  
- Services : commerces, services administratifs.  
- Analyses d’accessibilité et maillage territorial.

---

## 4. Grand Est Mobilité
- Flux de déplacements inter‑EPCI.  
- Matrices OD.  
- Déplacements domicile/travail, études, loisirs.  
- Intermodalité.

---

## <span style="color:red;">5. IGN</span> — Géoservices / BD TOPO
- Limites administratives.  
- Réseaux : routes, ferré, gares.  
- Toponymie.  
- Orthophotos, MNT/MNS.  
- Données SIG de référence.

---

## 6. Enseignement supérieur (partenaires)
- Effectifs par filière et site.  
- Alternants.  
- Évolutions annuelles.  
- Données isochrones via plugin spécifique.

---

## <span style="color:red;">7. DREAL / Data.gouv</span>
- Précarité énergétique (CSV/SHP).  
- Diagnostics régionaux.

---

## <span style="color:red;">8. Atmo GE</span>
- Indicateurs de qualité de l’air.  
- Polluants, mesures stationnelles.

---

## <span style="color:red;">9. DDT51</span>
- Dispositifs énergétiques : éolien, PV, méthanisation.  
- Risques naturels & technologiques.  
- Données réglementaires départementales.
